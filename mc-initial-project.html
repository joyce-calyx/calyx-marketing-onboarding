Production Field Guide

Calyx Owner: Marketing Coordinator Scope: Days 61–90 Internal Use
Overview

The Production Field Guide is the Marketing Coordinator's Initial Project — a structured reference document that maps end-to-end production workflows for Calyx's marketing materials. It exists so any marketer, new or established, can independently navigate a production request from brief to fulfillment without relying on tribal knowledge.

This project is not a simple task assigned to fill the calendar. It is a deliberate choice: the work forces the Marketing Coordinator to engage deeply with every function that touches production — from creative briefing and prepress through to Amie Hadley's fulfillment process — and to emerge from that investigation with documented, shareable systems.

The guide is scoped and kicked off within the first 90 days. It is not expected to be finished by Day 90. What matters at that checkpoint is that the scope is locked, contributors are aligned, and Phase 1 is actively in flight.

Objectives

The Production Field Guide has four objectives. Each maps to a gap the Marketing Coordinator will have already observed in the first 60 days of onboarding.

1
Eliminate production bottlenecks caused by knowledge gaps Map every step of the production request process — from brief submission through prepress, approval, fulfillment, and shipping — so no request stalls because someone doesn't know the next step.
2
Create a durable onboarding resource for future marketing hires Any future Marketing Coordinator or intern should be able to independently handle a production request within their first two weeks using this guide alone.
3
Surface and document the informal rules that govern production Calyx's production system runs on tribal knowledge — lead times, vendor preferences, file spec requirements, and fulfillment exceptions that live in people's heads. The guide pulls those into a single written record.
4
Demonstrate how the Marketing Coordinator scopes, plans, and ships work This project is also an evaluation instrument. How the MC approaches the scope, manages contributors, handles ambiguity, and hits milestones is evidence of how they'll manage real workstreams going forward.

Scope

What's in — and what's not
In scope
Marketing production request intake and ticketing · Prepress review and file specification standards · Internal approval workflow · Vendor and agency production hand-offs · Sample kit production process · Tradeshow materials production · Fulfillment and shipping process (Amie Hadley's workflow) · Lead times by material type · Common failure points and how to avoid them
Out of scope
Brand guidelines (maintained separately) · Creative strategy or campaign planning · Shopify/ecommerce product listings · Agency contract terms or vendor pricing · IT or systems procurement
